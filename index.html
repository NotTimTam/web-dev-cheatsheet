    Getting Started

    W.D.C. will not teach how to use HTML, CSS, and JS. It simply includes quick, copy-pastable code that can be used in any project.

    HTML

    Boiler-Plate

    The most basic starter format for an HTML document.

    <!DOCTYPE html>
<html lang="en">
<head>
		<eta charset="UTF-8">
		<meta name="viewport" content="width=device-width", initial-scale="1.0">
		<title>Document</title>
</head>
<body>
		
</body>
</html>

    Tags

    Text Tags

    The <!-- --> tags are used for comments.

    <h3>I am a Heading!<h3> <!-- 'h' tags are used for headers. They can be sizes numbered 1-6, with 1 being the main heading, and 6 being a minor heading. (i.e: h1, h2, h3, h4, h5, h6) -->
<p>Hello, World!<p> <!-- 'p' tags are used for simple text. -->

    Output:

    I am a Heading!
    

    Hello, World!

    Link Tags

    <a href="https://google.com/">link to google.com<a> <!-- 'a' tags are used for linking. They have a variety of options. This one will open in a new tab. -->

    Output:

    link to google.com
